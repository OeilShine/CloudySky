- 项目名称： 二手市场

- 项目介绍：

学生可以在平台上发布二手商品的消息，同时享受线上交易商品，发布商品交易并交易完成可增加用户在平台的信誉积分。该平台还引入了一些收购二手货物的企业和慈善机构，学生也可以将二手商品卖给企业或者通过该平台捐赠给慈善机构再由慈善机构捐赠给需要的人。(捐赠商品可换取相应积分，积分可兑换成小礼品或者兑换成相应的货币在改平台流通)。
